# TinderClone


## Technologies
- Frontend: React.js,Html,TailwindCss
- Backend: Node.js, Express.js, MongoDB


## Project Summary 

- This is a Tinder clone that allows fetching user profiles to either show interest or ignore.
- Built using Microservice Architecture.
- Includes Edit User Profile functionality.
- For password security, Password Hashing is implemented.
- For authentication, JWT tokens are used to store user ID in cookies.
